## Run Locally

Clone the project

```bash
  git clone git@github.com:al-alec/look4job_portal.git ./my-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
